POST /carts
DESCP: Create a content in a cart with variation
BODY
{
    "user_id": "321",
    "product_id": "111",
    "has_variation": true,
    "variation": {
        "quantity": 3,
        "color_index": 0,
        "size_index": 1
    }
}

DESCP: Create a content and update item in a cart without variation

    "quantity": 2,
    "user_id": "321",
    "product_id": "111",
    "has_variation":false

DESCP: To add note to cart

     "user_id":"321",
